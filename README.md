# ReUseTech

Marketplace simplificado para compra e venda de eletrônicos usados.

## Objetivo

O ReUseTech é um projeto MVP inspirado em plataformas como OLX e Mercado Livre, com foco em uma experiência enxuta para cadastro de usuários, listagem de produtos, carrinho simples e fluxo de checkout.

## Stack inicial planejada

- C# e .NET em backend.
- SQL Server.
- Dados mockados no front (somente em desenvolvimento)
- HTML
- Tailwind CSS.
