# ReUseTech

Marketplace simplificado para compra e venda de eletronicos usados.

## Objetivo

O ReUseTech e um projeto MVP inspirado em plataformas como OLX e Mercado Livre, com foco em uma experiencia enxuta para cadastro de usuarios, listagem de produtos, carrinho simples e fluxo de checkout.

## Stack inicial planejada

- Node.js
- Express
- SQLite no desenvolvimento
- PostgreSQL como evolucao para producao
- HTML
- Tailwind CSS via CDN
